LinQedin

University Project

LinQedin is an university project developed in C++ programming language, with the aim of forming a minimal system for the administration and use by way of graphical user interface of database of professional contacts inspired linkedin. The system provides to sign in as an administrator or as an client.
An LinQedin user is characterised by:

  • Profile (example: personal data, work experience, language, skills....)
  • Type of account registration: Basic, Business or Executive
  • Network of professional contacts

Client Side

It provides a GUI for the use linQendin services by registered user. The available features are:

  • Profile Update
  • Add new contacts to its own network
  • Remove contacts from its own network
  • User's search within linQedin DB, according to its own type of account.
    Three type of account provides search features growing:
    Basic allows a search that checks if exists inside DB with a specific name or surname;
    Business provides to get from a search the full profile of a user;
    Executive,also,allows to get the network of contacts of a user

Admin Side

It provides a GUI for the management of of the user database by administrator. The available features are:

  • Add new user in the DB
  • Remove users from DB
  • User's search within linQedin DB, through a name or a surname
  • Update type of account
  • Save and Read database information
  • Date: July 2015
  • Category: University Project

QuizziPedia

University Project

QuizziPedia is a Software Web that allows the creation and management of questionnaires able to detect the gaps of candidates before,during and at the end of a training course. The system offers the following features:

  • Store questionnaires in a server divided by topic;
  • Give the user, via an interface, specific questionnaires chosen topic;
  • Check and evaluate the questionnaires chosen by users according to the answers given.
  • Date: July 2016
