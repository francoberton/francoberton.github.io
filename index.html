Portfolio

Linqedin

Progetto universitario svolto con lo scopo di sviluppare in C++/Qt, un sistema minimale per l’amministrazione ed utilizzo tramite interfaccia utente grafica di un database di contatti professionali ispirato a LinkedIn.

Un utente LinqedIn è caratterizzato da:

  • Profilo (ad esempio: dati personali, esperienze professionali, competenze lavorative, lingue conosciute, titoli di studio, etc.).
  • Tipologia dell’account di iscrizione: Basic, Business o Executive.
  • Rete di contatti professionali con altri utenti Linqedin.

La mainwindow prevede la possibilità di scegliere di accedere come client oppure come admin

Lato admin

Prevede una GUI per la gestione basilare del database degli utenti LinqedIn da parte dell’amministratore. Le funzionalità disponibili sono:

  • Inserimento nel DB di un nuovo utente LinqedIn; i dati per l’inserimento saranno nome, cognome, codice univoco dell’utente (ad esempio, l’email), tipologia di account (Basic, Business, Executive).
  • Ricerca nel DB di un utente mediante nome/cognome o codice utente.
  • Rimozione dal DB di un utente LinqedIn.
  • Cambio di tipologia di account (Basic, Business, Executive) per un utente LinqedIn.
  • Salvataggio/lettura su file del DB degli utenti LinqedIn.
Lato client

Prevede una GUI per l’utilizzo dei servizi LinqedIn da parte di un utente già iscritto a LinqedIn. Le funzionalità disponibili sono:

  • Aggiornamento del proprio profilo.
  • Inserimento di un nuovo contatto nella propria rete.
  • Rimozione di un contatto dalla propria rete.
  • Funzionalità di ricerca sul DB LinqedIn, come autorizzate dalla propria tipologia di account. Le tre tipologie di account permettono delle funzionalità di ricerca crescenti: Basic permette una ricerca che controlla solamente se nel DB LinqedIn esiste un utente con un dato nome e/o cognome; Business permette di ottenere da una ricerca il profilo completo di un utente; Executive permette inoltre di ottenere anche la rete dei contatti di un utente.
